Como usar:

Compilar tudo:
make

Executar o servidor:
make run-server

Iniciar o servidor HTTP no diretório site/:

Caso o diretório não exista, crie-o:
mkdir site
echo "<h1>Servidor ativo!</h1>" > site/index.html

Acesse no navegador (pode ser em outro dispositivo na mesma rede):
http://<seu_ip>:8080/


Executar o cliente 

make run-client
